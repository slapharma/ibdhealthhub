Content Kit — What Gets Generated

📸 Instagram
  • Caption (2200 chars) + hashtags
  • Ava hook archetype applied
  • CTA (Grow/Engage/Convert/Save)
  • Reel script: HOOK / BODY / CTA
  • Static image (4:5) via Gemini Flash
  • 15s video (9:16) via FAL Kling
🎵 TikTok
  • Caption (300 chars) + 3-5 hashtags
  • Hook-first, casual tone
  • Reel script: HOOK / BODY / CTA
  • 15s video (9:16) via FAL Kling
💼 LinkedIn
  • Post (1300 chars), professional tone
  • Data-backed insight lead
  • Cover image (1:1) via Riverflow
  • Posts API — text + image
𝕏 X / Twitter
  • Thread of 3 posts (280 chars each)
  • Concise, punchy, 1-2 hashtags
  • Twitter v2 API
📘 Facebook
  • Post (2000 chars) + question CTA
  • Conversational, shareable
  • Cover image (1:1) via Gemini Flash
  • Graph API — text + image
📧 Substack
  • Newsletter teaser (600 chars)
  • Compelling summary + read-more CTA
  • Copy-to-clipboard only (no API)

Ava's Framework — Baked Into Generation

Content Pillar (auto-tagged)
🎓 Educate — clinical insight, authority building
🎭 Entertain/Relate — human story, behind the scenes
💰 Sell — testimonial, result, offer reveal
70% Educate/Entertain → 30% Sell enforced by queue
Hook Archetype (per post)
🕳️ Curiosity Gap
💥 Bold Claim / Contrarian
🔢 List / Number
🤝 Relatability
👆 Direct Call-Out
CTA System (one per post)
📈 Grow — "Follow for more [value]"
💬 Engage — "Comment [word] below"
🔗 Convert — "DM me / link in bio"
🔖 Save/Share — "Save this for later"
Auto-Schedule (Ava's optimal times)
Instagram: 7am, 12pm, 7pm
LinkedIn: 9am Tue–Thu
X/Twitter: 8am, 12pm, 5pm
Facebook: 9am, 1pm
Distributed across next 7 days on approval

Backend — Single Catch-All (stays within Vercel 12-function limit)

api/social/[...slug].js ← one function, routes internally
├── POST /generate ← LLM + image + video generation
├── GET /kits ← list all content kits
├── GET /kits/:id ← get one kit
├── PATCH /kits/:id ← edit text, approve platform
├── POST /post ← fire a scheduled post to platform API
└── GET /schedule ← pending scheduled posts

lib/social/handlers/ ← handler files per route
lib/social/platforms/ ← LinkedIn.js, Twitter.js, Instagram.js…
lib/social/media.js ← OpenRouter image + FAL.ai video
lib/social/ava-prompts.js ← Ava's hooks, CTAs, pillar prompts
lib/social/scheduler.js ← auto-schedule logic
