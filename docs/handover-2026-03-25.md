## Known Issues / Gotchas

### Stop Hook False Positive
The hookify/everything-claude-code plugin fires `[Preview Required] Code was edited but no dev server is running` after every edit. **This does not apply to this project** — there is no local dev server; verification is done by deploying to Vercel. The hook needs to be disabled for this project specifically.

### MiniMax Model ID
`minimax/minimax-m1:extended` is used (not `:free`). If this model stops working, check https://openrouter.ai/minimax for the current free model ID.

### Vercel Function Count
Currently using ~9 of 12 allowed serverless functions on the Hobby plan. Any new `api/*.js` files will count toward this limit. Keep new logic in `lib/` and route through the existing catch-all.
